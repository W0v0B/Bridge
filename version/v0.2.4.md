# v0.2.4

## Features

### 1. Collapsible sidebar and Quick Commands panel

**Implementation:** Added a collapse/expand toggle button at the bottom of the sidebar. When collapsed, the sidebar shrinks from 200px to 48px, showing only device status dots with tooltips. The Sider width transitions smoothly with CSS animation.

The Quick Commands panel in the Shell tab can also be hidden via a toggle button (double-arrow icon) in the terminal input area. When hidden, the shell terminal expands to fill the full width.

### 2. tlogcat for OHOS HiLog panel

**Implementation:** Added tlogcat streaming support to OHOS devices, matching the ADB LogcatPanel's dual-mode (logcat/tlogcat) design. The HiLog panel now has a Segmented control to switch between "HiLog" and "tlogcat" modes. Each mode runs independently per device with separate buffers.

Backend: New `start_tlogcat` and `stop_tlogcat` functions in `hdc/hilog.rs` that run `hdc -t <key> shell tlogcat` and stream output via `hdc_tlogcat_lines` events. Stderr lines are emitted as error-level entries with tag `tlogcat-stderr`.

Frontend: New `useHdcTlogcatEvents` hook, `startHdcTlogcat`/`stopHdcTlogcat` utility wrappers, and per-device-per-mode buffer management in HilogPanel (same pattern as ADB LogcatPanel).

### 3. Sortable Name and Modified columns in File Manager

**Implementation:** Added `sorter: true` to the "Name" and "Modified" columns in both ADB FileManager and OHOS HdcFileManager. Clicking a column header toggles between ascending/descending sort. Default sort is ascending by Name. Directories are always sorted before files regardless of sort direction. Name sorting uses `localeCompare` with `numeric: true` for natural ordering (e.g., `file2` before `file10`).

### 4. Quick access paths in File Manager

**Implementation:** Added a row of clickable path tags above the breadcrumb in both ADB FileManager and OHOS HdcFileManager. Users can:
- Click a tag to navigate to that path instantly
- Click the "+" tag to add a new quick access path (pre-fills current path)
- Close (×) a tag to remove it

Quick access paths are stored in the persisted config store (`adbQuickPaths` / `ohosQuickPaths`), so they survive app restarts. Default paths: ADB has sdcard/system/data; OHOS has data/system.
