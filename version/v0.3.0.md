# v0.3.0

## Features

### Screen Mirror (scrcpy integration)

Mirror and control ADB device screens directly from DevBridge by integrating scrcpy as an external subprocess. Each connected device gets an independent scrcpy window.

**Backend** (`src-tauri/src/adb/scrcpy.rs` — new file):
- `scrcpy_path()` — resolves scrcpy binary location: bundled `resources/scrcpy/scrcpy.exe` → Scoop/Chocolatey install paths → bare `"scrcpy"` on PATH
- `start(serial, config, app)` — spawns scrcpy with device serial and user-configured flags; tracks PID in a process-global `SCRCPY_PROCESSES` map; emits `scrcpy_state { serial, running: true }` event; background tokio task monitors process exit and emits `running: false` on termination
- `stop(serial, app)` — kills scrcpy via `taskkill /F /T /PID`; emits `scrcpy_state { serial, running: false }`
- `is_running(serial)` — checks PID registry

**ScrcpyConfig** — 15 configurable options:
- Display: max resolution, video bitrate, max FPS, crop, lock orientation
- Window: borderless, always on top
- Device: stay awake, show touches, turn screen off, power off on close
- Input: keyboard mode (uhid/sdk/aoa/disabled), mouse mode, disable audio
- Recording: record to file path

**Auto-cleanup on device disconnect**: The device watcher in `device.rs` now detects when a device disappears from `adb devices` and automatically kills its scrcpy process. This handles unexpected disconnects (cable pull, reboot), not just UI-initiated disconnects.

**Frontend** (`src/components/adb/ScreenMirrorPanel.tsx` — new file):
- Start/Stop button with loading state synced via `scrcpy_state` events
- Collapsible settings panel with 5 sections (Display, Window, Device, Input, Recording)
- Settings persisted to `localStorage` key `"scrcpy_config"`
- "Browse..." button for recording file path via native save dialog
- Error message with install link when scrcpy is not found

**New "Screen Mirror" tab** added to the ADB tab bar (between Logcat and File Manager).

**Note**: scrcpy is detected on PATH (not bundled) to keep the installer small. Users must install scrcpy separately from https://github.com/Genymobile/scrcpy.
