# v0.2.7

## Features

### Serial terminal Ctrl+C interrupt button

**Problem:** Serial port devices had no way to interrupt a running command (e.g., `ping`). Unlike ADB/OHOS devices which have a "Stop" button that kills the subprocess, serial ports are raw byte streams with no process to kill.

**Fix:** Added a `Ctrl+C` button next to the serial input area. When clicked, it sends the `0x03` byte (ETX / Ctrl+C) over the serial port — the standard interrupt signal that the device's shell interprets as SIGINT. The button is always visible for serial devices since commands run on the remote device, not locally.

## Fixes

### PanelGroup warnings when toggling Quick Commands

**Problem:** Hiding the Quick Commands panel triggered two console warnings: (1) `"Invalid layout total size: 70%"` because the main panel kept `defaultSize={70}` even when it was the only panel (should total 100%). (2) `"Panel id and order props recommended when panels are dynamically rendered"` because conditionally rendered panels lacked `id` and `order` props.

**Fix:** Added `id` and `order` props to both panels and the `PanelGroup`. Changed the main panel's `defaultSize` to `100` when Quick Commands is collapsed.
